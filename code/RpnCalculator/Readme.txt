
DATA ENTRY

H or h		Toggle between hexadecimal (64-bit) and floating-point mode
Digit 0-9	Numeric entry
Digit A-F	Numeric entry in hexadecimal mode
. or ,		Decimal point in floating-point mode
`			Change sign (+ or -)
E or e		Enter power-of-ten exponent in floating-point mode
I or i		Toggle between entry of real and imaginary parts
Enter		Finish (if entering) or duplicate the current number
Backspace	Correct an entry
Esc			Clear entry and stack
P or p		Enter pi

STATISTICS

Spacebar	Add sample
m			Average (mean)
M			Show sample count
v			Population standard deviation
V			Sample standard deviation
R or r		Generate uniformly distributed random number on [0,1)
g			Generate random number with standard Gaussian distribution

ROUNDING AND NUMBER PARTS

[		Round number toward zero
{		Round number toward nearest integer
]		Round toward negative infinity
}		Round toward positive infinity
"		Fractional part
		Real part
		Imaginary part
'		Absolute value
?		Sign (-1, 0, or +1)

BINARY OPERATIONS

~		NOT
&		AND
|		OR
^		XOR
<		Shift y left by x bits
>		Shift y right by x bits
		Rotate y left by x bits (hexadecimal mode only)
		Rotate y right by x bits (hexadecimal mode only)

BASIC FUNCTIONS

+		Add
-		Subtract
*		Multiply
/		Divide
%		Mod (remainder of division)
\		Reciprocal (1/x)
@		Square
Q or q	Square root
y		Raise y to the power of x
Y		Take the x-th root of y

TRIGONOMETRIC FUNCTIONS

s		sin
S		arcsin
Ctrl+S	sinh
		arcsinh
o		cos (letter "o")
O		arccos (Shift+letter "O")
Ctrl+O	cosh
		arccosh
t		tan
T		arctan
Ctrl+T	tanh
		arctanh

OTHER FUNCTIONS

x		Raise e to the power of x
X		Raise 10 to the power of x
Ctrl+X	Raise 2 to the power of x
l		Natural logarithm
L		Base-10 logarithm
Ctrl+L	Base-2 logarithm
N or n	Base-x logarithm
:		cos x + i sin x
;		arctan (Re(x) / Im(x)) (angle to positive real axis)
!		Factorial
W or w	Binomial--count combinations of y things taken x at a time
G		Gamma
Ctrl+G	Log Gamma

FUNCTION KEYS

Shift-Fn	Store current value in function key n
Ctrl+R		Begin or cancel recording of keystrokes
Ctrl-Fn		Store recorded keystrokes in function key n
F1-F12		Recall value of function key or replay keystrokes
